Інструкції
Це проєкт за змістом 1-го розділу курсу.
ЗАВДАННЯ: написати програму трансляції команд асемблерної програми в машинні коди (власне асемблер) та інструкцію до використання. Для написання асемблера треба обрати підмножину мови таку, щоб можна було складати деякі закінчені програми. Цей проєкт фактично є продовженням і розширеною версією лабораторної роботи 4, де були надані готові класи програмного коду C#, які можна використати.
Вибір підмножини мови (список команд, формати, способи адресування) виконати самостійно в будь-якому відносно функціонально повному варіанті. Обирати надто багато команд і способів адресування не треба. Писати асемблер можна або для 16-розрядного режиму, або для 32-розрядного, а запускати на виконання в обидвох випадках треба через емулятор - як в лабораторній роботі 4.
ОКРЕМІ ЗАУВАЖЕННЯ:
1) щоб приклади програм були гнучкими, потрібно включити до списку обраних до трансляції команди читання з клавіатури і друкування на екран, які виконують як переривання INT - треба уважно вивчити раніше наданий приклад для трансляції у файлі ex02_com.asm;
2) рекомендуємо писати асемблер для com-програм, цей варіант є простіший;
3) обов'язково скласти декілька тестових прикладів асемблерних програм різного змісту і з різними наборами команд і режимів адресування;
4) оформити проєкт краще як віконну програму з меню вибору операцій;
5) додати інструкцію для користувача до роботи з асемблером - як виконувати кожну операцію і додаткові особливості.
